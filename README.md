# 📝 Notes - Кроссплатформное приложение для заметок

Полнофункциональное мобильное приложение для создания и управления заметками с поддержкой изображений, избранного и гибкой системой сортировки. Разработано на **C# с использованием .NET MAUI**.

---

## 📋 Описание

**Notes** — это интуитивное и удобное приложение для работы с заметками, которое позволяет пользователям:
- Создавать и редактировать заметки с текстом и заголовками
- Прикреплять множество изображений к каждой заметке
- Добавлять заметки в избранное для быстрого доступа
- Быстро искать заметки по названию
- Персонализировать внешний вид с помощью встроенных тем
- Сортировать заметки по дате и названию

Приложение использует локальное хранилище (JSON-файл) для сохранения всех данных, обеспечивая полную автономность без необходимости интернета.

- **Просмотр изображений** - полноэкранный просмотр с каруселью
- **Удаление изображений** - удаление из просмотра
- **Автосохранение** - предложение сохранить при возврате
- **Отслеживание изменений** - уведомление о несохраненных данных

- **Отфильтрованный список** только избранных заметок

- **Удаление из избранного** одним нажатием
- **Быстрое удаление** заметок через свайп

## 🔧 Технологии и архитектура

### Основные технологии
- **C#** - язык программирования
- **.NET 9.0** - фреймворк
- **MAUI (Multi-platform App UI)** - кроссплатформенный UI-фреймворк
- **Syncfusion MAUI** - компоненты для работы с изображениями

### Архитектура
Проект использует **MVVM-подход** с четкой организацией кода:

```
Notes/
├── View/                  # Пользовательские интерфейсы
│   ├── MainPage.xaml(.cs)        # Главная страница со списком заметок
│   ├── NotePage.xaml(.cs)        # Редактор заметок
│   ├── FavoritesPage.xaml(.cs)   # Страница избранного
│   ├── ImagePage.xaml(.cs)       # Просмотр и редактирование изображений
│   └── SettingsPage.xaml(.cs)    # Настройки приложения
├── Model/                 # Модели данных
│   └── Note.cs           # Класс заметки с поддержкой PropertyChanged
├── Services/             # Бизнес-логика
│   └── NoteService.cs    # Работа с сохранением/загрузкой данных
├── Helpers/              # Утилиты и конвертеры
│   └── FavoriteIconConverter.cs   # Конвертер для смены иконки звезды
├── Resources/            # Ресурсы (изображения, шрифты)
├── Platforms/            # Платформоспецифичный код
├── Properties/           # Метаданные проекта
├── App.xaml(.cs)         # Корневое приложение и глобальные стили
├── AppShell.xaml(.cs)    # Структура навигации (Tab Bar)
├── MauiProgram.cs        # Конфигурация MAUI
└── Notes.csproj          # Конфигурация проекта
```

### Основные компоненты

#### Model: `Note.cs`
```csharp
public class Note : INotifyPropertyChanged
{
    public string Id { get; set; }           // Уникальный идентификатор (GUID)
    public string Title { get; set; }        // Название заметки
    public string Content { get; set; }      // Основной текст
    public bool IsFavorite { get; set; }     // Статус избранного
    public DateTime CreatedAt { get; set; }  // Дата создания
    public List<string> Images { get; set; } // Пути к изображениям
    public string DisplayTitle { get; set; } // Вычисляемое свойство для отображения
}
```

#### Services: `NoteService.cs`
- **GetNotesAsync()** - загрузка всех заметок из локального хранилища
- **SaveNotesAsync()** - сохранение заметок в JSON-файл
- Путь: `FileSystem.AppDataDirectory/notes.json`

#### Навигация: `AppShell.xaml`
- **TabBar** с тремя вкладками:
  1. Заметки (MainPage)
  2. Избранное (FavoritesPage)
  3. Настройки (SettingsPage)

#### Обмен событиями: MessagingCenter
- **NotesChanged** - событие изменения заметок
- **FavoritesChanged** - событие изменения избранного
- **SortChanged** - событие изменения сортировки

## 🚀 Инструкция по запуску

### Требования
- **Visual Studio 2022** или **Visual Studio Code**
- **.NET 9 SDK** с поддержкой MAUI
- **Android SDK** (для разработки на Android)
- **Java Development Kit (JDK) 11+**

### Установка и запуск

#### 1. Клонирование репозитория
```bash
git clone https://github.com/mekashik/Notes.git
cd Notes
```

#### 2. Восстановление зависимостей
```bash
dotnet restore
```

#### 3. Запуск на Android эмуляторе (через Visual Studio)
```bash
# Откройте проект в Visual Studio и нажмите F5
# Или используйте командную строку:
dotnet build -f net9.0-android -c Debug
dotnet run -f net9.0-android
```

#### 4. Создание APK для распространения
```bash
dotnet publish -f net9.0-android -c Release
# APK будет находиться в bin/Release/net9.0-android/publish/
```

#### 5. Запуск с помощью dotnet CLI
```bash
# Debug режим
dotnet run -f net9.0-android -c Debug

# Release режим
dotnet run -f net9.0-android -c Release
```

### Первый запуск
- Приложение создаст файл `notes.json` в `AppDataDirectory` при первом запуске
- Все заметки будут сохраняться автоматически

## 🔄 Ключевые алгоритмы

### Автосохранение заметки
```csharp
protected override bool OnBackButtonPressed()
{
    // Проверка наличия изменений
    if (!_hasChanges)
    {
        return true;
    }
    
    // Запрос подтверждения
    DisplayAlert("Сохранить заметку?", "Вы хотите сохранить изменения?", "Да", "Нет");
}
```

### Синхронизация между вкладками
```csharp
MessagingCenter.Subscribe<NotePage>(this, "NotesChanged", async _ =>
{
    _isLoaded = false;  // Флаг для перезагрузки
    await LoadNotesAsync();
});
```

### DisplayTitle - автоматическое название
```csharp
public string DisplayTitle
{
    get
    {
        if (!string.IsNullOrWhiteSpace(Title))
            return Title;  // Если есть название
        
        if (!string.IsNullOrWhiteSpace(Content))
            return Content.Split(' ').FirstOrDefault() ?? "Без названия";  // Первое слово из текста
        
        return "Без названия";  // По умолчанию
    }
}
```

## 🎯 Примеры использования

### Создание новой заметки
1. Нажмите кнопку **+** на главном экране
2. Введите название и текст
3. Добавьте изображения (опционально)
4. Нажмите **Сохранить**

### Добавление в избранное
1. На главном экране нажмите на звезду ☆ рядом с заметкой
2. Звезда превратится в ⭐
3. Заметка появится во вкладке "Избранное"

### Изменение сортировки
1. Перейдите в **Настройки**
2. Выберите нужный вариант сортировки
3. Главный экран обновится автоматически

### Удаление заметки
1. На главном экране проведите пальцем влево по заметке
2. Нажмите красную кнопку **Удалить**
3. Подтвердите удаление
